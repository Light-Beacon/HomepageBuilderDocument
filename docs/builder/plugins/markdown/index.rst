Markdown 插件
==============
Markdown 插件提供将 markdown 文档转换为 WPF 流文档的能力。

使用
--------------
.. tip:: 该插件默认安装并启用，你无须手动启用该插件

在导入卡片时，插件会读取后缀名为 `.md` 的文件并在构建器中注册以其文件名（不含后缀）为名称的卡片，卡片会将文件的属性加入卡片的对应字段中，并将文件正文的 markdown 代码加入卡片的 `markdown` 字段中。+

关于如何使用 markdown，请自行搜索教程。

你可以对这类卡片使用 `MarkdownCard` 模版来展示内容。你也可以 :doc:`继承<../template/index>` 该模版创建你自己的模版，你也可以使用脚本 `$MarkdownPresenter` 作为组件，它会生成一个内含卡片正文的流文档阅读器（ `FlowDocumentViewer` ）。

支持性
~~~~~~~~~~~~~~
插件并未支持所有 markdown 语法，但也支持一些 markodwn 本体中没有的语法。下表展示了对各类语法的支持程度

+-------+-------+---------+
| 语法    | 支持度   | 是否为扩展语法 |
+=======+=======+=========+
| 粗体    | ✅ 支持  | 否       |
| 斜体    | ✅ 支持  | 否       |
| 标题    | ✅ 支持  | 否       |
| 无序列表  | ✅ 支持  | 否       |
| 有序列表  | ✅ 支持  | 否       |
| 引用区块  | ✅ 支持  | 否       |
| 行内代码  | ✅ 支持  | 否       |
| 代码区块  | ✅ 支持  | 否       |
| 链接    | ✅ 支持  | 否       |
| 图片    | ✅ 支持  | 否       |
| 表格    | 尚未支持  | 否       |
| LaTeX | 无支持计划 | 否       |
| Html  | 无支持计划 | 否       |
| 关系图   | 无支持计划 | 否       |
| 维基链接  | 支持    | 是       |
| 警报    | 支持    | 是       |
+-------+-------+---------+
