脚本
==========
脚本是构建器内容、库属性、卡片列表中可以以代码调用的特殊占位符，当将脚本占位符放在内容中，在构建器进行格式化时读取到，构建器会运行脚本，脚本会根据当前的卡片、环境、脚本参数等运行并将输出内容替换为占位符。

占位符格式与属性占位符类似，但在最开始名称前会加上 `@` 以标示本占位符会调用脚本。

::

    ${@脚本名称|<必要参数>|...|[可选参数]|...}

脚本格式化后，会输出卡片名称（卡片引用），构建器会在此生成卡片名称为脚本输出内容的卡片
