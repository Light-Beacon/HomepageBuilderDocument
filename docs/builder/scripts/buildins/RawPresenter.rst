RawPresenter
=====================

.. automodule:: homepagebuilder.modules.Presenters
    :undoc-members:

简介
-------------
获取卡片的 `data` 属性，将其直接放入 xaml 代码中

使用
-------------
在模版中列表中使用该脚本

例：

.. code-block:: yaml

    components:
        - $RawPresenter
