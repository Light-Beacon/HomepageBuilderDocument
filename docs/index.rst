关于 Homepagebuilder
=============================

.. image:: https://repository-images.githubusercontent.com/738343500/aff4abf1-24e0-46a0-ab2d-14113cdfbfb7

Homepagebuilder 是一个PCL主页构建器框架，从基本的 XAML 或者其它比如 Markdown 类似的代码生成一个用于PCL的主页文件，以方便复杂主页的维护
